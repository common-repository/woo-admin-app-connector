== Description ==

The <strong>ADMIN APP CONNECTOR</strong> plugin will help the administrator to handle all the task from our mobile app. The admin can manage the customers orders , products , categories from our mobile app.

[INSTRUCTIONS TO INSTALL THE ADMIN APP](https://phoeniixx.com/wp-content/uploads/2019/03/Woo-admin-app-connector-documentation.pdf) | [ANDROID ADMIN APP DOWNLOAD LINK](https://play.google.com/store/apps/details?id=com.phoeniixx.wooadmin)

= Features =

<h4>Dashboard</h4>
<ul>
<li> Displays orders, sales, top-sellers etc.</li>
<li> View trends of week, current month, last month and current year.</li>
<li> View sales report in the form of graph.</li>
</ul>

<h4>Orders</h4>
<ul>
<li> Quickly see list of all orders with order number, customers details, date and status.</li>
<li> Update status of an order.</li>
</ul>

<h4>Products</h4>
<ul>
<li> View complete details of your products with name, thumbnail, price etc.</li>
<li> Edit and update product name, category, price, description etc.</li>
<li> Can view inventory status of products.</li>
</ul>

<h4>Categories</h4>
<ul>
<li> View complete details of your categories with name, image.</li>
<li> Edit and update category name, description, parent etc.</li>
</ul>

<h4>Customers</h4>
<ul>
<li> Can see list of all your customers details.</li>
<li> Edit customer details.</li>
<li> And can delete the customers.</li>
</ul>

<h4>Shop</h4>
<ul>
<li> View Categories and associated products.</li>
<li> View Product details with name, price, short description and description.</li>
<li> View Product Reviews.</li>
<li> View Product image with image sliders.</li>
</ul>

<h4>Other Features</h4>
<ul>
<li> Admin can give the permissions from backend settings.</li>
<li> Admin can add the user from backend.</li>
</ul>

== Installation ==

1. Install [WordPress](https://wordpress.org/)
2. Go to your WordPress admin panel, and then to Plugins. Click on "Add New" and then upload the zip file of the plugin using the "Upload Plugin" button you can find on top of the screen.
